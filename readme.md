# Guiter Music Webapp

![Guitarmusic](https://github.com/PriyankaBtech/GuitarMusic_App/assets/109729930/927b9d30-04c2-40e7-a385-bef2dd8b3879)

## Technologies Used
- HTML
- CSS
- JavaScript

## Features

- Play, pause, and skip tracks
- Manage playlists
- Adjust volume using the slider

## Usage

- Click on the play button to start playback.
- Use the pause button to pause playback.
- Skip to the next track with the skip button.
- Adjust the volume using the slider.
- Create and manage playlists by clicking on the playlist icon.
